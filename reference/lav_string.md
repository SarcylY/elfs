# Generating lavaan-readable single-factor string from model specifications

Generating lavaan-readable single-factor string from model
specifications

## Usage

``` r
lav_string(lfs_name, lfs_cols)
```

## Arguments

- lfs_name:

  Name of latent factor

- lfs_cols:

  Column(s) loading onto latent factor

## Value

single-factor string readable by lavaan
