# Generating Concatenated APA-formatted tables of results

Generating Concatenated APA-formatted tables of results

## Usage

``` r
cfa.tab.multi(x, y, z = NULL, a = NULL, b = NULL, robust = FALSE)
```

## Arguments

- x, y, z, a, b:

  lavaan model object(s)

- robust:

  Whether robust RMSEAs are calculated. Defaults to FALSE.

## Value

data.frame object in APA format
